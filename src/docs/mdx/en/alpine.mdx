## Alpine Mirror Usage Guide

### URL

https://mirrors.jcut.edu.cn/alpine/

### Description

Alpine Linux software repository.

### Supported Architectures

x86_64, aarch64, loongarch64

---

### Usage Instructions

#### Method 1: One-line Replacement (Recommended)

Run the following command to replace the upstream URL in your existing `/etc/apk/repositories`:

```shell
sed -i 's/dl-cdn.alpinelinux.org/mirrors.jcut.edu.cn/g' /etc/apk/repositories
```

#### Method 2: Edit the Config File Directly

Select your Alpine version below, then **overwrite** the contents of `/etc/apk/repositories` with the generated output:

<ConfigGenerator promptString="Select your Alpine Linux version:" versionList={alpineVersionList} defaultVersion={defaultVersion} configGen={GenAlpineConfig} language="text" />

After copying, open the file with your editor:

```shell
sudo vi /etc/apk/repositories
# or
sudo nano /etc/apk/repositories
```

Replace the file contents with the generated config above, save, then run:

```shell
apk update
```

---

### Notes

`latest-stable` automatically points to the current latest stable release, which is suitable when you do not need to pin a specific version. If you want to prevent unexpected upgrades, select a specific version number (e.g. `v3.21`).

After modifying `/etc/apk/repositories`, always run `apk update` to apply the changes.
