### Usage Instructions

#### Method 1

Generally, it is sufficient to replace the default repository address `http://archive.ubuntu.com/` with `http://mirrors.jcut.edu.cn/` in the `/etc/apt/sources.list` file.

You can use the following command:

```shell
sudo sed -i 's|archive.ubuntu.com|mirrors.jcut.edu.cn|g' /etc/apt/sources.list
```

Starting from *Ubuntu 24.04 LTS*, DEB822 format is used instead of the traditional One-Line-Style format. The corresponding file to modify is `/etc/apt/sources.list.d/ubuntu.sources`. You can also use the following command:

```shell
sudo sed -i 's/archive.ubuntu.com/mirrors.jcut.edu.cn/g' /etc/apt/sources.list.d/ubuntu.sources
```
