## openEuler镜像使用帮助

### 地址

[https://mirrors.jcut.edu.cn/openeuler/](https://mirrors.jcut.edu.cn/openeuler/)

### 说明

openEuler 公共镜像软件源

### 收录架构

官方支持

### 使用方法

将`/etc/yum.repos.d/openEuler.repo`文件中默认的源地址 `http://repo.openeuler.org/` 替换为 `https://mirrors.jcut.edu.cn/openeuler/`

```bash
sed -i.bak 's|http://repo.openeuler.org/|https://mirrors.jcut.edu.cn/openeuler/|g' /etc/yum.repos.d/openEuler.repo
```
