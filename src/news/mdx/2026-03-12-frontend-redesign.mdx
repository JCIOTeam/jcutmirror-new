大家好，我是镜像站的管理员。

距离上一次镜像站页面更新已经过去很久了。之前的旧版页面虽然稳定，但随着大家频率变高，很多问题都暴露出来了，再加上多年来累积的一些技术债，维护起来也越来越吃力。

所以，趁着近期有空，我们决定对前端进行一次彻底的重构。今天，**JCUT Mirror** 新版本正式和大家见面了。

## 这次改了啥？

这次重构没有堆砌复杂的功能，主要还是围绕 **「好用」** 和 **「好看」** 两点：

1.  **终于支持完善的深色模式了**
知道大家晚上查文档、下代码多，刺眼的白光确实难受。现在页面右上角可以切换深色/浅色模式，设置会保存在本地，下次来访自动生效。
2.  **搜索更快了**
以前搜索可能需要跳转或刷新，现在是实时过滤。输入几个字母，镜像列表立刻缩小范围。同时还加了 A-Z 字母索引，找特定字母开头的软件更方便。
3.  **目录页风格统一**
这点可能大家平时注意不到，但以前点击具体镜像进入文件列表（FancyIndex）时，风格会突然变回原始 Nginx 页面。这次我们通过注入 header/footer，让文件列表页也能继承主站的主题和语言设置，体验更连贯。
4.  **帮助文档更好读了**
内置的文档现在支持 Markdown 渲染，代码块有高亮，还提供了一键复制按钮。

## 关于技术栈（给感兴趣的同学）

这次重构我们没用老式的 jQuery 或 Vue 2，而是直接上了比较新的技术栈，主要是为了后续维护更方便，性能也更好：

*   **核心**：React 18 + TypeScript
*   **UI**：Material UI v7
*   **构建**：Vite
*   **状态管理**：Zustand + TanStack Query

代码已经开源在 GitHub 上（许可证 GPLv3）。如果有其他高校镜像站想复用，或者本校同学想参与贡献，欢迎提 PR 或 Issue。我们也踩了不少坑（比如 MUI v7 的兼容性问题、FancyIndex 的状态同步等），希望能帮到后来者。
